Investigation Overview¶

In this project I performed an exploratory analysis on data provided by Ford GoBike, using Python visualization techniques. The goal is to find a spesific relationships between some variables that can provide the needed visualization between them .

Dataset Overview¶

This dataset includes information about individual rides made in a bike-sharing system covering the greater San Francisco Bay area.There are 519700 rows of trip trip in the dataset.

# load in the dataset into a pandas dataframe
fordgobike = pd.read_csv('2017-fordgobike-tripdata.csv')

#Change the dtypes.
fordgobike.start_time = pd.to_datetime(fordgobike.start_time)
fordgobike.end_time = pd.to_datetime(fordgobike.end_time)

fordgobike.bike_id = fordgobike.bike_id.astype(str)
fordgobike.user_type = fordgobike.user_type.astype('category')

# add some needed columns .
fordgobike['hour'] = fordgobike.start_time.dt.strftime('%H')
fordgobike['day'] = fordgobike.start_time.dt.strftime('%a')
fordgobike['month'] = fordgobike.start_time.dt.strftime('%b')
